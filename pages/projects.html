Static Website Deployment

This project involved deploying a static website using Amazon S3 (Simple Storage Service) buckets for hosting web content. The website consists of static HTML, CSS, and JavaScript files, making it an ideal candidate for deployment on S3, which supports highly scalable and durable storage.

Pawsitive

An app to save street dogs and cats by allowing users to report their locations via calls or GPS. The app enables community involvement in animal rescue and adoption, ensuring that reported animals are rescued, treated, and placed in loving homes.

This project demonstrates web technologies through a song-themed app, using HTML, CSS, Bootstrap, JavaScript, PHP, and MySQL for a dynamic, responsive experience. It includes user authentication, session management, and database integration. The project highlights modern web development practices, combining front-end and back-end technologies.
