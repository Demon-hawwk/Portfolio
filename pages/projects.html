Static Website Deployment Using Amazon S3 Buckets

This project involved deploying a static website using Amazon S3 (Simple Storage Service) buckets, providing a cost-effective and efficient solution for hosting web content. The website consists of static assets such as HTML, CSS, and JavaScript files, making it an ideal candidate for deployment on S3, which supports highly scalable and durable storage.
